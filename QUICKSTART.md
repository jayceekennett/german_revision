# QUICKSTART GUIDE

## Introduction
german-revision is run in terminal (macOS) or Command Prompt/PowerShell on Windows. It requires python to be installed. Once you have opened a terminal, run these prompts.
Note: these commands are case-sensitive. Make sure your working directory is the same folder that the game has been saved in. Use the command 'pwd'to check your working directory.


#### 1. Install python for your operating system
https://www.python.org/downloads/


#### 2. Check python is installed correctly:
Type:
*python3 --version*
You should see a version (e.g *Python 3.14.3*)


#### 3. Navigate to the repository
Type:
*cd path/to/german_revision_project*
(Change the filepath to the folder you've saved the repository to)


#### 4. Create a python environment
Type:
*python3 -m venv .venv*


#### 5. Activate the environments
On MacOS type:
*source .venv/bin/activate*

On Windows (Command Prompt):
*.venv\Scripts\activate*

On Windows (PowerShell):
*.venv\Scripts\Activate.*


#### 6. Install the project
Type:
*pip install .*


#### 7. Run the game:
Type:
*german-revision*


#### Returning to the game:
The environment must be activated. Use the same command as above based on your operating system. The commands below presume MacOS is being used.
Type:
cd path/to/german_revision
source .venv/bin/activate 
german-revision
