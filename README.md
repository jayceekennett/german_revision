# README

## Overview
This project provides a command-line German-revision game designed to support structured vocabulary and grammar practice. 
It allows users to test themselves in both German-to-English and English-to-German directions, drawing on CSV-based word lists. 
The game includes modules for practising core linguistic skills such as selecting the correct article for a noun, forming noun plurals, and conjugating German verbs. 

The nouns, verbs, adjectives, adverbs, and plural nouns were taken from freely available corpuses of German words and ordered by frequency.
conjugations.csv holds the top 2897 most frequently used verbs in German, downloaded from https://gist.github.com/wanderingstan/7eaaf0e22461b505c749e268c0b72bc4

## Version 2 features:
Version 2 includes the option to unscramble German sentences, practicing one of 6 German grammar rules:
- V2 (verb in second place)
- Time-manner-place word order
- Subject-object inversion
- Modal vebs
- Seperable verbs
- Perfect tense
- Subordinate clauses
The user can select to see the basic overview of these rules at any time with the ':rules' command.

Version 2 supports the InputHandler class to ensure input is parsed in a nomalised way and allowing commands via ":" at ant time.

This version also includes bug fixes and code-cleaning.
